POM Viewer
===========================================
Display pom.xml information in a jar file.

### How to install

     jenv selfupdate
     jenv install pomviewer 0.0.1

### How to use

     pom.sh xxxx.jar

### FAQ
How to install jenv? Please reference [https://github.com/linux-china/jenv](https://github.com/linux-china/jenv)
